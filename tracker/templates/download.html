Instructions d'installation détaillées

Windows (Le plus simple)

Fichier : run_p2p_windows.bat
  1. Téléchargez le fichier run_p2p_windows.bat
  2. Double-cliquez sur le fichier .bat
  3. Une fenêtre de commande s'ouvre et demande : 
    Mode d'installation :
1) Installation complète (Tracker + Peer)
2) Peer uniquement (se connecter à un tracker existant)
Votre choix [2] :
  4. Choisissez "2" (Peer uniquement - pour se connecter à notre tracker)
  5. Le script demande l'URL du tracker : 
    URL du Tracker [https://p2p-file-sharing-production.up.railway.app] :
    ➜ Appuyez simplement sur Entrée (l'URL par défaut est déjà configurée)
  6. Le script demande le port du peer : 
    Port du Peer [8001] :
    ➜ Appuyez sur Entrée (utiliser le port par défaut)
  7. ✅ C'est tout ! Le système :
    • ✅ Installe Python (si nécessaire)
    • ✅ Crée un environnement virtuel
    • ✅ Installe les dépendances
    • ✅ Démarre le peer
    • ✅ Ouvre automatiquement le navigateur sur http://localhost:8001
Résumé Windows :
Télécharger → Double-cliquer → Choisir "2" → Entrée × 2 → ✅ Terminé !

Linux

Fichier : run_p2p_linux.sh
  1. Téléchargez le fichier run_p2p_linux.sh
  2. Ouvrez un terminal dans le dossier où le fichier est téléchargé
  3. Rendez le script exécutable :
    chmod +x run_p2p_linux.sh
  4. Exécutez le script :
    ./run_p2p_linux.sh
  5. Le script demande le mode d'installation : 
    Mode d'installation :
1) Installation complète (Tracker + Peer)
2) Peer uniquement (se connecter à un tracker existant)
Votre choix [2] :
    ➜ Tapez "2" et appuyez sur Entrée
  6. Le script demande l'URL du tracker : 
    URL du Tracker [https://p2p-file-sharing-production.up.railway.app] :
    ➜ Appuyez sur Entrée (l'URL par défaut est déjà configurée)
  7. Le script demande le port : 
    Port du Peer [8001] :
    ➜ Appuyez sur Entrée (utiliser le port par défaut)
  8. ✅ C'est tout ! Le système démarre automatiquement
  9. Ouvrez votre navigateur et allez sur :
    http://localhost:8001
Résumé Linux :
chmod +x run_p2p_linux.sh && ./run_p2p_linux.sh → Choisir "2" → Entrée × 2 → Ouvrir http://localhost:8001

macOS

Fichier : run_p2p_linux.sh (même script que Linux)
  1. Téléchargez le fichier run_p2p_linux.sh
  2. Ouvrez Terminal (Applications → Utilitaires → Terminal)
  3. Naviguez vers le dossier Téléchargements :
    cd ~/Downloads
  4. Rendez le script exécutable :
    chmod +x run_p2p_linux.sh
  5. Exécutez le script :
    ./run_p2p_linux.sh
  6. Suivez les mêmes étapes que Linux (choisir "2", appuyer sur Entrée 2 fois)
  7. Ouvrez Safari ou Chrome et allez sur http://localhost:8001
Résumé macOS :
Terminal → cd ~/Downloads → chmod +x run_p2p_linux.sh && ./run_p2p_linux.sh → Choisir "2" → Entrée × 2

Ce que fait automatiquement le script :

  • ✅ Vérifie que Python est installé
  • ✅ Crée un dossier ~/.p2p_file_sharing (ou %USERPROFILE%\.p2p_file_sharing sur Windows)
  • ✅ Télécharge tous les fichiers nécessaires
  • ✅ Crée un environnement virtuel Python
  • ✅ Installe toutes les dépendances
  • ✅ Configure la connexion au tracker
  • ✅ Démarre le peer en arrière-plan
  • ✅ Crée des scripts helper (stop_p2p.sh, restart_p2p.sh)
