Fonctionnalités

Détection Python, création environnement virtuel, installation dépendances
Fragmentation distribuée
Fichiers volumineux fragmentés et distribués sur plusieurs peers
Sécurisé
Vérification d'intégrité avec SHA-256
Téléchargement parallèle
Récupération simultanée depuis plusieurs sources
Réseau P2P
Communication directe entre peers sans serveur central
Interface Web
Dashboard complet accessible depuis le navigateur

Instructions d'installation

  1. Téléchargez le script correspondant à votre système d'exploitation
  2. Rendez le script exécutable (Linux/macOS) ou exécutez-le (Windows)
  3. Suivez les instructions à l'écran pour configurer votre peer
  4. Accédez au dashboard via http://localhost:8001
  5. Partagez vos fichiers et commencez à utiliser le réseau P2P !
